Element

Create buttons, alerts, generally apply this style where you see fit.

Simple usage with the @include element;, and beyond to more complex usage @include element-base; @include element-shading;. Built with reusability, customisability and speed. The current style button should always produce great results no matter what colour you throw at it. Should you want to change the style and I encourage it, you should copy the element-shading mixin into a new file and make changes.

Detailed usage ▾

@include element(
	$main-color: $alt, // any colour
	$text-color: $white, // any colour
	$radius: $radius, // any radius, false for none
	$subtle: false, // fully only styled on hover/active
	$shading: true, // remove gradients/shadows
	$size: $default-size, // adjust the size with a number 1, 2, 3, etc
	$hover: false, // add a hover state
	$font-size: false, // change font-size
	$display: inline-block, // change the display type
	$dry: false // use no repetitive code see alternative usage
);

Alternative usage ▾

// extend this
@include element-base(
	$size: $default-size, // adjust the size with a number 1, 2, 3, etc
	$font-size: false, // any size
	$text-color: $white, // any colour
	$subtle: false, // fully only styled on hover/active
	$radius: $radius, // any radius, false for nothing 
	$display: inline-block // change the display type
);
@include element($alt, $dry: true);
